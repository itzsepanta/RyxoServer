Quick Start

Launch in seconds

Get RyxoServer up and running with just a few commands.

Terminal 
git clone https://github.com/itzsepanta/RyxoServer.git
cd RyxoServer
npm install
npm run start:dev

SSL/HTTPS Setup

Create certs/ folder, add server.key & server.crt, then set SSL_ENABLED=true in .env.
